# Правила орфографии в Java

Названия классов пишутся с большой буквы, пример:
- HelloWorld
- Main
- Start
- CamelCaseMethode


Названия методов мы начинаем с маленькой буквы:
- main
- start
- add
- changeVolume
- camelCaseMethode
